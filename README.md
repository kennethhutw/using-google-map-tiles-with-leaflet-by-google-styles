# Leaflet.TileLayer.GoogleMap.Grayscale

A regular TileLayer with grayscale makeover.

## Usage

` var googleLayer = new L.Google('ROADMAP',{ styles:[{stylers: [{saturation: -100}]}]});
        map.addLayer(googleLayer);` .
